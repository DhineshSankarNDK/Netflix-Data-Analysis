
# Netflix Data Analysis

## Project Overview

This project analyzes Netflix Movies and TV Shows data using Python.

The project focuses on:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend Analysis

The dataset contains information about Netflix content such as:
- Movies
- TV Shows
- Genres
- Ratings
- Release Years
- Countries

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

Dataset used:
Netflix Movies and TV Shows Dataset

File:
```text
netflix_titles.csv
```

## Features

- Removed duplicate data
- Handled missing values
- Analyzed top genres
- Compared Movies vs TV Shows
- Visualized release year trends

## Key Insights

- Netflix has more Movies than TV Shows.
- Drama and International genres dominate the platform.
- Content production increased rapidly after 2015.
- TV-MA is one of the most common ratings.
- The United States contributes a large amount of Netflix content.
